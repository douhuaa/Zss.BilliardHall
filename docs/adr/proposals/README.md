## 提案状态说明

| 状态 | 说明 | 下一步 |
|------|------|--------|
| 📋 Proposal | 初步提案，待评审 | 团队讨论 → RFC |
| 🔍 Under Review | 正在评审中 | 架构委员会评审 |
| ✅ Approved | 已批准，待实施 | 转化为正式 ADR |
| ❌ Rejected | 已拒绝 | 存档 |
| ⏸️ On Hold | 暂停 | 等待前置条件 |

## 提交提案

如需提交新的 ADR 提案，请：

1. 使用模板：[ADR 提案模板](../../templates/adr-proposal-template.md)（如存在）
2. 在本目录创建新文件：`ADR-[主题]-Proposal.md`
3. 包含以下内容：
   - 问题陈述
   - 提案概述
   - 预期影响
   - 实施成本
   - 风险评估
4. 提交 PR 并标记相关审批人
