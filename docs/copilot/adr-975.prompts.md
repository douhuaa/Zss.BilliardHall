## 概述

本文件为 GitHub Copilot 提供关于文档质量监控的场景化指导。

**核心原则（来自 ADR-975）**：
- 持续监控文档质量
- 自动检测文档问题
- 提供质量指标
- 定期审计机制

---

## 场景 1：检查文档质量

### 用户问："如何检查文档质量？"

**Copilot 应该**：

```bash
文档质量检查：

1. 运行完整验证：
./scripts/verify-all.sh

2. 检查特定方面：
- 链接有效性
- 格式一致性
- ADR 完整性
- 版本同步

3. 查看质量报告：
docs/QUALITY-REPORT.md（如自动生成）
```
