## 概述

本文件为 GitHub Copilot 提供关于实践案例库的场景化指导。

**核心原则（来自 ADR-951）**：
- 案例库记录实践经验
- 案例不具备裁决力
- 必须引用对应 ADR
- 便于学习和参考

---

## 场景 1：创建新案例

### 用户问："如何记录一个实践案例？"

**Copilot 应该**：

```markdown
创建案例步骤：
1. 在 docs/cases/ 创建文件
2. 使用标准格式：
   - 背景说明
   - 引用相关 ADR
   - 解决方案描述
   - 代码示例
   - 经验教训

示例：
```
