## 概述

本文件为 GitHub Copilot 提供关于交互式学习路径的场景化指导。

**核心原则（来自 ADR-965）**：
- 提供交互式学习体验
- 分步骤、可验证的学习路径
- 即时反馈和验证
- 促进主动学习

---

## 场景 1：使用交互式学习

### 用户问："如何使用交互式学习路径？"

**Copilot 应该**：

```
交互式学习包括：

1. 动手练习：
   - 按步骤完成小任务
   - 运行测试验证
   - 获得即时反馈

2. 验证机制：
   - 运行架构测试
   - 检查代码符合规范
   - 获得通过/失败反馈

3. 渐进式学习：
   - 从简单到复杂
   - 每个阶段有明确目标
   - 完成一个再进入下一个
```
