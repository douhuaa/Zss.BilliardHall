## 场景 1：编写符合标准的脚本输出

### 用户问："如何让脚本输出符合日志标准？"

**Copilot 应该**：

```bash
标准日志格式：

# 成功
echo "✅ 操作成功：说明"

# 警告
echo "⚠️  警告：说明"

# 错误
echo "❌ 错误：说明"

# 信息
echo "ℹ️  信息：说明"

示例脚本：
#!/bin/bash
echo "ℹ️  开始验证..."
if [ 条件 ]; then
    echo "✅ 验证通过"
    exit 0
else
    echo "❌ 验证失败：原因"
    exit 1
fi
```
