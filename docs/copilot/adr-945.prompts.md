## 概述

本文件为 GitHub Copilot 提供关于 ADR 时间线和演进视图的场景化指导。

**核心原则（来自 ADR-945）**：
- 维护 ADR 时间线视图
- 记录 ADR 演进历史
- 可视化 ADR 发展路径
- 便于追踪架构演进

---

## 场景 1：查看 ADR 时间线

### 用户问："如何查看 ADR 的创建和演进时间线？"

**Copilot 应该**：

根据 ADR-945，ADR 时间线在以下位置：

```
docs/adr/ADR-TIMELINE.md - 自动生成的时间线视图

包含：
1. 按时间排序的 ADR 列表
2. 每个 ADR 的状态变更历史
3. 主要版本的里程碑
```

---

## 场景 2：追踪 ADR 演进

### 用户问："如何追踪一个 ADR 的演进历史？"

**Copilot 应该**：

```markdown
查看 ADR 演进：
1. 查看 ADR 文档中的"版本历史"章节
2. 查看 Git 提交历史：
   git log --follow docs/adr/ADR-XXXX.md
3. 查看关系图中的"替代"关系
```
