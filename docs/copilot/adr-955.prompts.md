## 概述

本文件为 GitHub Copilot 提供关于文档搜索和可发现性的场景化指导。

**核心原则（来自 ADR-955）**：
- 文档必须易于搜索
- 提供多种索引方式
- 维护文档间的交叉引用
- 优化文档可发现性

---

## 场景 1：查找相关文档

### 用户问："如何快速找到相关的 ADR 或文档？"

**Copilot 应该**：

```
查找文档的方法：

1. 查看 README 索引：
   - docs/README.md - 总索引
   - docs/adr/README.md - ADR 索引
   - docs/copilot/README.md - Prompts 索引

2. 使用关系图：
   - docs/adr/ADR-RELATIONSHIP-MAP.md

3. 按主题搜索：
   grep -r "模块隔离" docs/

4. 查看 Quick Reference：
   docs/QUICK-REFERENCE.md
```
